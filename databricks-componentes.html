biblioteca - boto3

O boto3 facilita a integração de sua aplicação, biblioteca ou script Python aos serviços da AWS, incluindo Amazon S3, Amazon EC2 e Amazon DynamoDB, entre outros. O Boto3 foi criado do zero para oferecer suporte nativo nas versões 2.7 ou posterior e 3.4 ou posterior do Python.
Sintaxe para importação : “import boto3”

biblioteca - requests

Otimiza o layout dos dados do Delta Lake. Opcionalmente, possibilita otimizar um subconjunto de dados ou otimizar por coluna. Exemplos de casos de uso :
Otimizando um dataset :
OPTIMIZE events
Otimizando dataset com condição :
OPTIMIZE events WHERE date >= '2017-01-01'
Utilizando ZorderBy :
OPTIMIZE events
WHERE date >= current_timestamp() - INTERVAL 1 day
ZORDER BY (eventType)
