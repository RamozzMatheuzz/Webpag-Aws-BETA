camada - raw

“Raw” - é a primeira etapa do processo do Delta Lake House, é onde armazenamos todos os dados em sua forma bruta, seja de uma só fonte ou várias, de diversas maneiras.

camada - bronze

“BRONZE” - é a segunda etapa do processo do Delta Lake House, é onde armazenamos os dados da Camada anterior, porém, transformados em Delta, sendo essa a única transformação até então.

camada - silver

“SILVER” - é a terceira etapa do processo do Delta Lake House, é o step onde já ocorre a transformação de dados (ETL), cujo propósito é fazer com que os dados já otimizados (Delta), se comportem no formato de Tabelas, semelhante a um DataBase
